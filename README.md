# AMR-SLAM

Implementation of SLAM algorithms on a customized TurtleBot3 Buger.

---

## 🛠 Hardware Specifications

| Component | Specification |
| :--- | :--- |
| **Base Platform** | **TurtleBot3 Burger** (Customized) |
| **Main Compute** | **NVIDIA Jetson Orin Nano** (8GB) *(Upgraded from RPi4)* |
| **Camera** | **Intel RealSense D435I** (RGB-D / IMU) |
| **Lidar** | **LDS-02** (360° Laser Distance Sensor) |
| **Embedded Controller** | OpenCR 1.0 |
| **Actuator** | **Dynamixel XL430-W250-T** (x2) |
